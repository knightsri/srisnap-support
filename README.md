## FAQ

**Q: How much does it cost to use?**  
A: The app is free. You provide your own OpenAI or OpenRouter API key. Typical usage costs $1-2/month.

**Q: Is my data stored?**  
A: No. Images are sent to the AI provider for analysis and immediately discarded. We don't have servers that store your data.

**Q: Which AI models are supported?**  
A: Any vision-capable model via OpenAI API or OpenRouter (GPT-4o, Claude, Gemini, etc.)

**Q: Can I use it offline?**  
A: No, an internet connection is required to process images through the AI.
